# Sky Skills

Custom Claude Code skills collection.

## Skills

| Skill | Description |
|-------|-------------|
| [linux-kernel-dev](skills/linux-kernel-dev/) | Linux kernel & driver development - coding standards, module/driver templates, Kconfig, Makefile, device tree, debugging, kernel API reference |

## Installation

```bash
# Register marketplace
/plugin marketplace add <your-github-user>/sky-skills

# Install
/plugin install linux-kernel-dev@sky-skills
```

Or install locally by adding the path to your Claude Code settings.
