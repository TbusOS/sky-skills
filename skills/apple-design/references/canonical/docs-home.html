REFERENCES 参考资料

What the skill ships. 这个 skill 里有什么。

Nine reference files, one reading order: tokens → typography → layouts → components. Each earns its place. 九份参考,一条阅读顺序:tokens → 字体 → 布局 → 组件。每一份都有用。

01

design-tokens.md

CSS variables for colour, spacing, radius, type scale. Components defer to tokens — no hard-coded hex. 颜色、间距、圆角、字号的 CSS 变量。组件全部走 tokens,禁止硬编码 hex。

QUICK START 快速上手

Use it in three lines. 三行就能用起来。

From a Claude chat 在 Claude 对话里

Prefix your request with apple 风格 (or any trigger word in SKILL.md). The skill activates automatically. Run the three-gate design-review before shipping. 在请求前加 apple 风格(或 SKILL.md 里任何触发词)。Skill 自动激活。发布前跑三道 design-review 闸。

Read the trigger list ›查看触发词列表 › 
# 1) Ask Claude in the apple voice
> apple 风格,写一张 Skypad pricing 页

# 2) Verify structure
$ python3 skills/design-review/scripts/verify.py \
    my-page.html

# 3) Render + audit + screenshot
$ bin/design-review --skill=apple \
    my-page.html

IN THE HARNESS 在 HARNESS 里

One of eight generators.
Evaluated by one discriminator. 八个生成器之一。
由一个判别器审核。

apple-design generates. design-review judges. The split is intentional — GAN-style. No generator evaluates its own output. apple-design 负责生成。design-review 负责判别。这个分离是有意的 —— GAN 式。没有生成器评判自己的输出。
