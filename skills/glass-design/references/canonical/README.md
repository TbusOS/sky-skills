# glass-design canonical library

Reference implementations. Each page-type ships as an `html` + `md` pair:
the HTML is the gold standard, the MD is the decision record + acceptance
rubric. Read the pair BEFORE generating a page of that type.

| Page-type | Status | What it proves |
|---|---|---|
| `landing.html` | v1 · 2026-06-11 | Aurora light budget, console mock in Tier 1 glass, solid-cyan brand trio, count-up markup-truth |
| `diagram-gallery.html` | v1 · 2026-06-11 | **Flagship** — engineering diagrams in the page's own material, 6 genres, theme-proof SVG ink |
| `dashboard.html` | v1 · 2026-06-11 | Calm aurora, KPI count-up row, hand-drawn streaming charts, mono event table |
| `data-report.html` | v1 · 2026-06-11 | Long-form prose on glass (R1), embedded charts, incident timeline |

All pages: dual-theme (`data-theme` dark canon + light variant, audited
both), bilingual (§G), self-diff block (§M), freeze-contract motion
(motion.md §0). Gates: `bin/design-review --skill=glass <page>` runs
dark + light automatically.
