Long-form prose 长文写作

Essays, weekly letters, drafts that need to live in plain Markdown. The blank page opens in 80ms; saves write directly to disk so the file is always recoverable from the filesystem, not from a cloud queue. 随笔、周更、需要以纯 Markdown 形式存在的草稿。空白页 80ms 打开；保存直接写入磁盘文件，可以从文件系统恢复，不依赖云端队列。

Used daily by writers at the Atlantic and a handful of arXiv-hosted essayists. 每天在《大西洋月刊》编辑部和若干 arXiv 写作者那里被用作主力。

Linked vault 关联笔记仓

Linked Markdown with citation autocomplete (BibTeX or arXiv ID), backlinks computed in 18ms, and per-file frontmatter that reads as plain YAML for any other tool. Markdown 文档互链，BibTeX / arXiv 引用自动补全，反向链接 18ms 算完，每篇 frontmatter 是标准 YAML——任何工具都能读。

Latency 延迟

< 12ms keystroke-to-render median, p99 21ms on M2 Air with a 100k-character buffer. Measured by bench/keystroke-loop over 50,000 chars of mixed prose + code. 击键到渲染中位 < 12ms，p99 21ms（M2 Air · 10 万字 buffer）。用 bench/keystroke-loop 跑 5 万字混合文本测得。

Storage 存储

Plain Markdown files in ~/Documents/Skypad/. Index DB at ~/.skypad/index.db (rebuildable from files at any time). Up to 50,000 entries tested without index degradation. 纯 Markdown 文件存 ~/Documents/Skypad/。索引库在 ~/.skypad/index.db（任何时候可从文件重建）。测过 5 万条不掉速。

Keyboard 键盘

QWERTY · Dvorak · Colemak · AZERTY · Workman, plus custom physical-key mapping via ~/.skypad/keys.toml. Chord cheatsheet identical across layouts. QWERTY · Dvorak · Colemak · AZERTY · Workman 五种，加 ~/.skypad/keys.toml 自定义物理键。和弦速查表所有布局一样。

Export 导出

Markdown · plain text · PDF (LaTeX-rendered) · JSON · plain HTML (no inlined CSS, you bring the stylesheet). All offline, no cloud round-trip. Markdown · 纯文本 · PDF（LaTeX 排版）· JSON · 纯 HTML（不内联 CSS，自己带样式表）。全部离线，不走云端。

Offline 离线

Fully functional. The only network call is optional sync (off by default). Plane mode safe. 完全可离线运行。仅在主动开启同步时有网络调用（默认关闭）。飞行模式安全。

Data policy 数据政策

Local-first. Zero telemetry, zero crash reports unless you opt in. Cloud sync (when enabled) is end-to-end encrypted with a key derived on your machine. 本地优先。不发任何遥测、不上传崩溃报告，除非你主动开启。云端同步（开启时）端到端加密，密钥在你机器本地派生。

Automation 自动化

CLI hooks via ~/.skypad/hooks.toml: pre-save · post-save · on-tag. Hooks run as your user, not as root. ~/.skypad/hooks.toml 配 CLI 钩子：pre-save · post-save · on-tag。以当前用户身份运行，不走 root。

Accessibility 无障碍

VoiceOver · NVDA · Orca tested on each release. Dyslexic-friendly font option, all UI reachable by keyboard, color-contrast passes WCAG AA throughout the app chrome. 每次发版都过 VoiceOver · NVDA · Orca 测试。提供阅读障碍友好字体选项，所有界面可全键盘到达，应用所有色彩对比度过 WCAG AA。

Honest limits 诚实的局限

Three things Studio is not the right tool for. 三件事 Studio 不是合适的工具。

Real-time collab is not native — for live multi-cursor work pick Notion, Google Docs, or Hedgedoc. Database-style views are intentionally absent — Studio is a notebook, not Airtable; if you need linked tables with formulas, use Obsidian DataView or a real database. Mobile editing is read-only — capture on phones happens through Skypad Reader, the sibling app, and syncs back to Studio when you're at your desk. 原生不支持实时协同——多人光标场景请选 Notion · Google Docs · Hedgedoc。我们也刻意没做数据库式视图——Studio 是笔记本，不是 Airtable；如果你要带公式的关联表，请用 Obsidian DataView 或真正的数据库。手机端是只读的——手机上的捕获走 Skypad Reader，回到桌面时再同步回 Studio。
