03 · Quick start快速上手

Use it in three lines. 三行就能用起来。

From a Claude chat 在 Claude 对话里

Prefix your request with sage 风格 (or any trigger word in SKILL.md). The skill activates automatically. Run the three-gate design-review before shipping. 在请求前加 sage 风格(或 SKILL.md 里任何触发词)。Skill 自动激活。发布前跑三道 design-review 闸。

Read the trigger list查看触发词列表 
# 1) Ask in the sage voice
> sage 风格,写一张 Skypad landing 页

# 2) Verify structure
$ python3 skills/design-review/scripts/verify.py \
    my-page.html

# 3) Render + audit + screenshot
$ bin/design-review --skill=sage \
    my-page.html
04 · In the harness在 harness 里

One of eight generators.
Evaluated by one discriminator. 八个生成器之一。
由一个判别器审核。

sage-design generates. design-review judges. The split is intentional — GAN-style. No generator evaluates its own output. sage-design 负责生成。design-review 负责判别。这个分离是有意的 —— GAN 式。没有生成器评判自己的输出。
