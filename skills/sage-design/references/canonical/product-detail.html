02 · The Studio windowStudio 主窗口

One window, three panes. 一扇窗，三栏。

Vault on the left, prose in the middle, backlinks on the right. Nothing ever opens over your text — both side panes fold away with ⌘ E. 左边是仓库，中间是正文，右边是反向链接。没有任何东西会盖在文字上面——两个侧栏用 ⌘ E 一键收起。

03 · Use cases用途

Three jobs it does well. 三件它做得好的事。

Studio was designed for these. If your work isn't on this list, read the honest limits further down before you start the trial. Studio 当初就是为这些场景设计的。如果你的写作不在这张清单上，开始试用之前请先读下面的「诚实的局限」。

Long-form drafts长文草稿

Essays, chapters, weekly letters. The blank page opens in 80 ms and every save writes straight to disk — the draft is recoverable from the filesystem, not from a cloud queue. 随笔、章节、周更。空白页 80ms 打开，每次保存直接写进磁盘——草稿从文件系统就能恢复，不依赖云端队列。

Research notes研究笔记

Linked Markdown with citation autocomplete — BibTeX or arXiv ID. Backlinks are computed in 18 ms, and frontmatter is plain YAML that any other tool can read. Markdown 互链，引用自动补全——支持 BibTeX 或 arXiv ID。反向链接 18ms 算完，frontmatter 是任何工具都能读的纯 YAML。

Daily log每日日志

⌘ N from anywhere opens today's note in the daily folder. One file per day, dated filenames, and yesterday is always one backlink away. 在任何地方按 ⌘ N，就打开 daily 文件夹里今天的笔记。一天一个文件，文件名带日期，昨天永远只隔一条反向链接。

04 · Specifications规格

The numbers, plainly. 数字，照实说。

Six rows a tester could verify in an afternoon. When a number changes between releases, this table changes first. 六行规格，一个下午就能逐条核实。版本之间哪个数字变了，先改的就是这张表。

Storage format存储格式

Plain Markdown files in ~/Documents/Skypad/. The index lives at ~/.skypad/index.db and can be rebuilt from the files at any time. Tested to 50,000 notes without slowdown. 纯 Markdown 文件，存在 ~/Documents/Skypad/。索引在 ~/.skypad/index.db，任何时候都能从文件重建。测到 50,000 条笔记不掉速。

Search latency搜索延迟

⌘ K full-text search returns in under 90 ms on a 50,000-note vault. Keystroke-to-render stays under 12 ms median, p99 21 ms, on an M2 Air with a 100k-character buffer. ⌘ K 全文搜索在 50,000 条笔记的仓库上 90ms 内返回。击键到渲染中位数低于 12ms，p99 21ms（M2 Air，10 万字 buffer）。

Shortcuts快捷键

Three chords cover the day — ⌘ N new note, ⌘ K switcher, ⌘ E writing mode; the full palette is six keys. Layouts: QWERTY · Dvorak · Colemak · AZERTY · Workman, custom mapping via ~/.skypad/keys.toml. 三个和弦覆盖一天——⌘ N 新建，⌘ K 跳转，⌘ E 写作模式；完整面板共六个键。键盘布局：QWERTY · Dvorak · Colemak · AZERTY · Workman，自定义映射写在 ~/.skypad/keys.toml。

Versions kept版本保留

Every save is a local snapshot, stored as plain Markdown diffs in ~/.skypad/history/. Pro keeps 90 days of history; on Free, history trims to the last 7 days. 每次保存都是一个本地快照，以纯 Markdown diff 形式存在 ~/.skypad/history/。Pro 保留 90 天历史；免费版保留最近 7 天。

Import & export导入与导出

Import: point Studio at any folder of .md files — it opens as a vault, nothing is converted. Export: Markdown · plain text · PDF (LaTeX-rendered) · Epub · JSON · plain HTML. All offline, no cloud round-trip. 导入：把 Studio 指向任何一个装 .md 文件的文件夹，它直接作为仓库打开，不做任何转换。导出：Markdown、纯文本、PDF（LaTeX 排版）、Epub、JSON、纯 HTML。全部离线完成，不经过云端。

Encryption加密

Sync, when you turn it on, is end-to-end encrypted with a key derived on your machine — servers only ever see ciphertext. Zero telemetry. An open-source security audit is published every six months. 同步在你主动开启时才运行，端到端加密，密钥在你的机器上派生——服务器只见过密文。零遥测。每六个月发布一次开源安全审计。

05 · Honest limits诚实的局限

What Studio is not. Studio 不是什么。

Three limits we would rather you read before the trial, not after it. 三条局限，我们希望你在试用之前读到，而不是之后。

01

No real-time co-editing. Two cursors in one note is not what Studio is for. For live collaboration, Notion, Google Docs, or Hedgedoc will serve you better. 没有实时协同编辑。一篇笔记里出现两个光标不是 Studio 的方向。要实时协作，Notion、Google Docs 或 Hedgedoc 都比它合适。

02

No database views, no web clipper. Studio is a notebook, not Airtable and not a read-it-later inbox. For linked tables with formulas, use Obsidian DataView or a real database. 没有数据库视图，也没有网页剪藏。Studio 是笔记本，不是 Airtable，也不是稍后读收件箱。要带公式的关联表，请用 Obsidian DataView 或真正的数据库。

03

Mobile is a read-mostly companion. Phones browse and capture through Skypad Reader; editing waits until you are back at the desk, then syncs in. 手机端是以读为主的伴侣应用。手机上用 Skypad Reader 浏览和捕获；编辑等你回到桌面再做，然后同步回来。

Studio in the lineup. 三件套里的 Studio。
