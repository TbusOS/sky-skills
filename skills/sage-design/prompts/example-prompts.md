# Ember Design 示例 Prompt

## 基础

```
用 ember-design 风格做一个小众咖啡工坊的首页，
hero 用 Fraunces serif 大标题，棕色 CTA 按钮，金色点缀。
```

## 文学博客 / 期刊

```
用 ember-design 排版一篇 1800 字的人物专访：
720px 单栏 Inter 正文，中间穿插 Fraunces italic pull quote（带金色大引号），
· · · 分节符，底部带作者信息。
```

## 产品落地页（手作品牌）

```
用 ember-design 做一个独立香水品牌 'Ember & Oud' 的着陆页：
- hero: 品牌名 + 一句话定位 + 两个按钮（探索 / 购买）
- 3 款招牌产品的卡片
- 创始人引用
- 深色 dark section 放品牌哲学
- 底部订阅表单
```

## 精品酒店

```
用 ember-design 做一家山中精品酒店的预订页：
房型卡、价格表、客户评价轮播、预订表单（棕色提交按钮）。
```

## 色板套用

```
按 ember-design 色板（#fff2df 底 + #312520 文字 + #492d22 CTA + #c49464 金）
重写这段 HTML，保留结构只改样式。
```

## 不要这样用

- ❌ 赛博朋克 / 深色 UI
- ❌ 高科技 SaaS 着陆页（用 apple-design）
- ❌ 研究 / 论文 / 白皮书风格（用 anthropic-design）
