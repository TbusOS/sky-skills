Frequently asked

Can I switch plans mid-cycle?

Yes. Upgrades take effect immediately and we prorate the difference. Downgrades apply at the next billing cycle so you keep what you have already paid for.

What counts as a "message"?

Each user turn that produces a Claude response counts as one message. Tool calls and intermediate thinking tokens do not count separately.

Is API usage included?

Pro, Max, Team, and Enterprise plans include $5 of API credits per seat per month. Additional API usage is billed at standard API rates.

How does Anthropic handle my data?

By default we do not train models on data from paid plans. See our privacy policy for the full breakdown.
