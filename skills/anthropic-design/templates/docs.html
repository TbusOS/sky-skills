Messages API

Send a structured list of input messages and receive the model's next message in the conversation. The Messages API is the primary surface for building assistants, agents, and analysis pipelines on Claude.

Request

Send a POST to /v1/messages with a JSON body containing the model, messages array, and a max_tokens ceiling. The example below uses the Python SDK. 

from anthropic import Anthropic

client = Anthropic()

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Summarize this quarter's earnings call."}
    ],
)

print(response.content[0].text)
Tip

For prompts that exceed 1,024 tokens and are reused across requests, enable prompt caching with cache_control: {"type": "ephemeral"} on the system message. Cached reads are 90% cheaper than uncached input.

Response shape

A successful response includes id, model, role, an array of typed content blocks, and a usage object with input and output token counts.
