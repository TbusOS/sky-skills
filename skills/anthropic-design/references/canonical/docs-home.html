Render like anthropic.com. 写一次。

This skill makes Claude output HTML in anthropic's editorial voice — warm cream background, Poppins + Lora pairing, orange accent, italic pull-quotes. Pair it with a trigger word, or reference it by name, and the generator will write in this style. 这个 skill 让 Claude 用 anthropic 的编辑风格输出 HTML —— 暖米底、Poppins + Lora 配对、橙色点缀、斜体引语。在 prompt 里用触发词或直接点名,生成器就会按这个风格写。

What the skill ships. 这个 skill 里有什么。

Reading order: tokens → typography → layouts → components. Each one earns its place in the final output; they're not decorative. 阅读顺序:tokens → 字体 → 布局 → 组件。每一份都在最终产出里起作用,不是装饰。

01

design-tokens.md

CSS variables for colour, spacing, radius, type scale. Every component defers to tokens — no hard-coded hex. 颜色、间距、圆角、字号的 CSS 变量。组件全部走 tokens —— 禁止硬编码 hex。

02

typography.md

Poppins (heading / display) + Lora (italic body quotes) + JetBrains Mono (code). Sizes, weights, leading, and when to use each. Poppins(标题 / 展示)+ Lora(斜体引文)+ JetBrains Mono(代码)。字号、字重、行高,以及各自的使用场景。

layout-patterns.md

Six layout skeletons: marketing hero, pricing grid, blog long-form, doc page, feature-split, full-bleed showcase. Plus the container selection rule. 六种布局骨架:营销 hero、定价网格、博客长文、文档页、双栏特性、全宽展示。附 container 选择规则。

components.md

28 reusable blocks: nav, buttons, badges, cards, quotes, banners, footers, SVG illustrations. With class= and minimal HTML per block. 28 个可复用组件:导航、按钮、徽章、卡片、引文、横幅、页脚、SVG 插画。每个组件附 class 与最小 HTML。

Quick start快速上手

Use it in three lines. 三行就能用起来。

From a Claude chat 在 Claude 对话里

Prefix your request with an anthropic 风格 (or any trigger word in SKILL.md). The skill activates automatically. Then run the three-gate design-review before shipping. 在请求前加上 anthropic 风格(或 SKILL.md 里任何触发词)。Skill 自动激活。发布前跑三道 design-review 闸。

Read the trigger list查看触发词列表 
# 1) Ask Claude in the anthropic voice
> anthropic 风格,写一张 Skypad pricing 页

# 2) Save to a file, then verify
$ python3 skills/design-review/scripts/verify.py \
    my-page.html

# 3) Render + audit + screenshot
$ bin/design-review --skill=anthropic \
    my-page.html
In the harness在 harness 里

One of eight generators.
Evaluated by one discriminator. 八个生成器之一。
由一个判别器审核。

anthropic-design generates. design-review judges. The split is intentional — GAN-style. No generator evaluates its own output. anthropic-design 负责生成。design-review 负责判别。这个分离是有意的 —— GAN 式。没有生成器评判自己的输出。
