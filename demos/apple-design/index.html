Installation

One line per skill.

Skills install through Claude Code CLI or drop-in symlinks. Each is isolated — adopt just the ones you need. 

# via Claude Code
claude install github:TbusOS/sky-skills/skills/apple-design

# or, drop-in
git clone https://github.com/TbusOS/sky-skills.git
ln -s "$(pwd)/sky-skills/skills/apple-design" \
  your-project/.claude/skills/apple-design
Note. Skills auto-activate based on the TRIGGER keywords in their frontmatter. You don't invoke them manually.
