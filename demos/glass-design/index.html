02 · How it works工作方式

A folder, a keyword, a verdict. 一个文件夹、一个关键词、一个判决。

A folder is a skill.一个文件夹就是一个技能。

Each skill is a self-contained directory: SKILL.md for the contract, references for the craft, assets for the CSS, templates to copy from. No registry, no build — delete the folder and it's gone. 每个技能都是一个自带全部家当的目录：SKILL.md 写契约，references 放工艺文档，assets 放 CSS，templates 给你直接复制。没有注册表、没有构建——删掉文件夹，它就消失了。

Anatomy of a skill解剖一个技能

TRIGGER picks the moment.TRIGGER 挑准时机。

Every SKILL.md carries TRIGGER keywords in its frontmatter. Say "liquid glass" and this skill loads itself — its tokens, its canon, its gates — without a single manual import. 每份 SKILL.md 的 frontmatter 里都写着 TRIGGER 关键词。你说出 liquid glass，这个技能就自己加载进来——它的 token、它的范本、它的检查——不需要任何手动引入。

What is a skill什么是技能

Generators ship, gates judge.生成的归生成，评审的归评审。

Five design skills generate pages; design-review judges them — structure, rendered audit, screenshots, then an LLM critic. A page that fails any gate is a page that doesn't ship. 五个设计技能负责生成页面，design-review 负责评判——结构检查、渲染审计、截图、再加一个 LLM 评审。任何一道检查不过，页面就不算完成。

Code architecture, an SoC block and a three-remote git flow — the same trio every sibling demo draws, here in dark glass: nodes are panels, lines are hairlines, cyan is where the energy goes. 代码架构、SoC 框图、三远端 git 协作——每个姊妹 demo 都画的同一组图，这里换成暗玻璃语言：节点是面板，线是发丝线，cyan 指向能量去的地方。

05 · Generator / evaluator生成器 / 评审器

Skills generate.
Gates judge. 技能负责生成。
检查负责评判。

The five generators carry strong opinions, so the judges are deliberately mechanical. Three scripts and one critic stand between any page and the public directory — and this very page just went through all four. 五个生成器都带着明确的审美立场，所以评审刻意做成机械的。任何页面和公开目录之间隔着三个脚本和一位评审——你正在看的这页也刚刚过完这四道。

G·1

verify.py — structure, staticallyverify.py——静态查结构

Placeholders, undefined classes, bilingual coverage, full-width punctuation — caught before a browser ever opens. 占位符、未定义的 class、双语覆盖、全角标点——浏览器还没打开就先拦下。

G·2

visual-audit.mjs — render, then measurevisual-audit.mjs——渲染后逐项测量

Playwright renders the page and measures it: WCAG contrast, diagram sizing, hollow cards, brand presence. Glass pages run twice — dark and light. Playwright 渲染页面后逐项测量：WCAG 对比度、图示尺寸、空心卡、品牌占比。glass 页面要跑两遍——dark 和 light。

G·3

screenshot.mjs — a human looksscreenshot.mjs——人眼把关

Full-page screenshots in both themes, reviewed before any commit. Taste the scripts can't measure, a person still can. 全页截图、双主题，提交前必看。脚本量不出来的品位，人看得出来。

G·4

critic — taste, scoredcritic——给品位打分

An LLM critic scores composition, brand, copy and craft. Below 88, the issues feed the next round instead of the repo. LLM 评审给构图、品牌、文案、工艺打分。低于 88 分，意见进下一轮，页面进不了仓库。

06 · Install安装

One folder. Zero build. 一个文件夹，零构建。

Clone once, symlink anywhere. Claude activates a skill only when your prompt hits its TRIGGER words — installing all fourteen costs nothing until you ask. 克隆一次，到处符号链接。只有提示词命中 TRIGGER 词，Claude 才会激活对应技能——把十四个全装上，不开口就没有任何成本。 

# clone once, symlink anywhere
git clone https://github.com/TbusOS/sky-skills.git
ln -s "$(pwd)/sky-skills/skills/glass-design" \
  your-project/.claude/skills/glass-design
