Philosophy理念

Not every page wants to shout.不是每个页面都想大声说话。

Tech-company web design tends toward two modes: crisp Apple minimalism or energetic Anthropic warmth. Both are excellent — but they leave out a whole category of brand: the handcraft one. The boutique hotel. The small-batch roaster. The essayist’s independent journal. The perfumery that wants to feel like a letter from 1892. 科技公司的网页设计往往落在两种模式里：利落的 Apple 极简，或是热情的 Anthropic 暖调。两者都很好 —— 却漏掉了一整类品牌：手作的那一类。精品旅店。小批量烘焙的咖啡铺。散文作者的独立期刊。想让人觉得像一封 1892 年来信的香水铺。

ember-design is that third mode. A warm cream page. Chocolate text. A rich brown button that feels like leather. A gold hairline under every link, like gilt on a book. Fraunces for display — a variable serif whose softness axis you can actually dial in. Inter for comfortable reading. It is unhurried on purpose. ember-design 就是那第三种模式。暖奶油色的页面。巧克力色的文字。一颗摸上去像皮革的深棕按钮。每个链接下一道金色细线，像书页上的烫金。标题用 Fraunces —— 一款柔软度真的可以调的可变衬线体。正文用 Inter，读起来舒服。它的从容是有意为之。

The best interfaces don’t announce themselves. They feel like a room you’ve been in before. 最好的界面不会宣告自己的存在。它们像一个你来过的房间。

Install安装

One line per skill.每个技能，一行命令。

Either Claude Code’s CLI, or a plain symlink. Skills are just folders — nothing magic. 用 Claude Code 的命令行，或者一条普通软链接。技能就是文件夹 —— 没有魔法。 

# via Claude Code
claude install github:TbusOS/sky-skills/skills/ember-design

# or, symlink
git clone https://github.com/TbusOS/sky-skills.git
ln -s "$(pwd)/sky-skills/skills/ember-design" \
  your-project/.claude/skills/ember-design
