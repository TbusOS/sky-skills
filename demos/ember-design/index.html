Philosophy

Not every page wants to shout.

Tech-company web design tends toward two modes: crisp Apple minimalism or energetic Anthropic warmth. Both are excellent — but they leave out a whole category of brand: the handcraft one. The boutique hotel. The small-batch roaster. The essayist’s independent journal. The perfumery that wants to feel like a letter from 1892.

ember-design is that third mode. A warm cream page. Chocolate text. A rich brown button that feels like leather. A gold hairline under every link, like gilt on a book. Fraunces for display — a variable serif whose softness axis you can actually dial in. Inter for comfortable reading. It is unhurried on purpose.

The best interfaces don’t announce themselves. They feel like a room you’ve been in before.

Install

One line per skill.

Either Claude Code’s CLI, or a plain symlink. Skills are just folders — nothing magic. 

# via Claude Code
claude install github:TbusOS/sky-skills/skills/ember-design

# or, symlink
git clone https://github.com/TbusOS/sky-skills.git
ln -s "$(pwd)/sky-skills/skills/ember-design" \
  your-project/.claude/skills/ember-design
Auto-trigger

Claude activates the skill only when your prompt matches the TRIGGER words in its frontmatter. No manual invocation.
