Philosophy

Not every page wants to shout.

Apple’s crispness and Anthropic’s editorial warmth cover a lot of ground. Ember adds the handcraft lane. But there’s still a room left unlit: the quiet one. The reading app. The botanical studio. The modern journal that wants generous margins and no fireworks. The software brand that trusts the user to come closer rather than waving to catch attention.

sage-design is that room. Rice-paper cream. Sage green accents that feel like a houseplant on a shelf, not a highlight marker. Deep indigo for display. Instrument Serif italic for the headline, Inter for everything you actually read. Generous negative space between ideas. It is restrained on purpose — because confidence doesn’t need volume.

The best interfaces don’t announce themselves. They feel like a room you’ve been in before.

Install

One line per skill.

Either Claude Code’s CLI, or a plain symlink. Skills are just folders — nothing magic. 

# via Claude Code
claude install github:TbusOS/sky-skills/skills/sage-design

# or, symlink
git clone https://github.com/TbusOS/sky-skills.git
ln -s "$(pwd)/sky-skills/skills/sage-design" \
  your-project/.claude/skills/sage-design
